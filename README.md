# 🔐 Modern Authentication System (Python GUI)

A clean and modern **Login & Registration GUI application** built using **Python** and **CustomTkinter**.
This project demonstrates how to design a **modern desktop authentication interface** with a smooth dark theme and user-friendly layout.

The application allows users to switch between **Login** and **Register** screens with a simple and elegant interface.

---

## ✨ Features

✔ Modern **Dark Mode UI**
✔ Built using **CustomTkinter** for better visuals
✔ Separate **Login and Registration screens**
✔ Smooth frame switching
✔ Clean and minimal design
✔ Beginner-friendly Python project

---

## 🖥️ Application Preview

The application includes two main interfaces:

### Login Screen

* Username input field
* Password input field
* Login button
* Option to create a new account

### Registration Screen

* Username field
* Email field
* Password field
* Sign-up button
* Back to login option

## ⚙️ Installation

Follow these steps to run the project locally.

### 1️⃣ Clone the Repository

```bash

```

### 2️⃣ Navigate to the Project Folder

```bash

```

### 3️⃣ Install Required Library

```bash
pip install customtkinter
```

### 4️⃣ Run the Application

```bash
python app.py
```

## 📸 How It Works

1. The application starts with the **Login screen**.
2. Users can enter a username and password.
3. If the user doesn't have an account, they can click **Create Account**.
4. The **Registration screen** appears where new users can sign up.
5. The interface switches dynamically between screens.
